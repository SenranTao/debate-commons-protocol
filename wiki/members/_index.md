# 成员目录

本页面列出全部 DCP 成员及其当前 Badge 状态。成员分为两类平行轨道：

- [[individuals/_index|Individual（个人）]] — 以个人身份加入
- [[communities/_index|Community（团体）]] — 社团、赛事组织、SPAR 群等

> **验证 Badge**：任何第三方可通过搜索成员 ID 确认其 Badge 状态。当前在线地址为 [GitHub Pages URL 待发布]。

## 状态说明

| Badge 状态 | 含义 |
|-----------|------|
| Active | 良好信誉，履行义务 |
| Probation | 信任受损，部分权益暂停 |
| Terminated | 协议终止，成员资格丧失 |
| Banished | 永久除名，不可逆转 |

完整规则见 [[protocol/constitution|章程]] 第七条及 [[protocol/operations|运营手册]] 第三章。
