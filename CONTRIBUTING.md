# 贡献指南 (CONTRIBUTING)

> 辩论信息公地协议 (DCP) — 投稿指南

DCP 是一个去中心化的资源共享网络。你贡献的每一份资源都会被纳入共享 Database，供整个网络的其他成员使用。以下是投稿的基本规则。

---

## 投稿渠道

协议尚未正式上线。在接入渠道（网页表单等）建好之前，投稿可以：
- 直接联系 Founding Maintainer（DCP-0GGS）
- 或关注协议的后续通知

---

## 我能贡献什么

### 原创资源

| 类型 | 说明 | 示例 |
|------|------|------|
| motions | 辩题库 | 比赛辩题汇总、按赛事整理 |
| methods | 训练方法 | 练习设计、技战术分析、学习路线 |
| transcripts | 比赛 transcript | 辩论赛文字转写、录音整理 |
| ai-tools | AI 工具使用 | Prompt 模板、AI coach 使用经验 |

### 非原创资源整理（curatorial）

你可以筛选、索引、注释网络上的优质辩论资源。你的贡献在于**整理和注释**，而非将原作者内容据为己有。

| 类型 | 说明 |
|------|------|
| curated-video | 非原创视频（YouTube 教学、比赛录像等） |
| curated-audio | 非原创音频（播客、讲座录音等） |
| curated-text | 非原创文字（文章、研究报告等） |
| english-learning | 英语学习资源 |
| news | 辩论议题相关新闻 |
| casefiles | 辩题案例文件、证据卡片 |

---

## 投稿格式要求

### 基本要求

1. **内容完整**：你提交的材料结构清晰，不残缺
2. **标注身份**：附上你的 DCP 成员 ID 或名称
3. **正面贡献**：投稿的目的应当是为辩论学习提供价值

### Curatorial 资源额外要求

对于非原创整理的资源，**必须**标注：
- 原始来源（URL / 作者 / 发布日期）
- 原始版权归属
- 你的筛选理由和注释（为什么这份资源值得看？）

**如果缺少来源标注，投稿将被退回。**

**如果直接复制他人原创内容且无整理/注释增值，投稿将被退回。** 全文引用等于搬运，整理+注释才是贡献。

---

## 质量评估

每份投稿 AI 会进行质量检查，判断通过或不通过。

通过标准：内容完整、结构清晰、有用、格式规范。Curatorial 投稿还需来源标注完整。

不通过 → 退稿，附改进建议。改好了可以重新投。

详细规则见 [运营手册](wiki/protocol/operations.md)。

---

## 许可

所有投稿以 **CC BY-SA 4.0** 许可发布。这意味着：
- 任何人使用你的内容时，必须注明你的名字和协议信息页链接
- 任何基于你内容的衍生作品，必须以相同的 CC BY-SA 4.0 许可发布

更多规则见 [章程](wiki/protocol/constitution.md) 第三条第 3.3 款。

---

## 成员责任

每位成员对其上传至共享 Database 的内容负全部责任。如果上传了无权授权的内容——例如未经许可转载他人的版权材料——由上传者自行承担法律后果。
