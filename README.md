# Debate Commons Protocol (DCP)

> 辩论信息公地协议 · Debate Commons Protocol
>
> 去中心化的大学生英语辩论资源共享网络

---

## 这是什么

DCP 不是一个组织。它是一个**标准**——一套自愿订阅的规则，让辩论社群的信息共享不再是损失，不共享才是。

核心理念来自 Richard Stallman 的 Copyleft（GPL 传染机制 → 辩论资源共享）、Elinor Ostrom 的公地治理、B-Corp 的认证机制。

## 怎么用

- **辩手**：订阅协议 → 将资源贡献到共享 Database → 获得网络内所有成员群组的置顶互通 + Preferential Access
- **社团/赛事/SPAR 群**：订阅协议 → 在自己运营的群置顶协议资源中心 → 赛事信息触达所有其他成员群组
- **任何人**：浏览本仓库的 LLM-Wiki，查阅成员目录、共享资源、协议文本——全部公开，无需订阅

## 仓库结构

```
wiki/
  members/      — 成员目录（每位成员一个页面，含 Badge 状态）
  protocol/     — 章程、运营手册、规则修改提案
  resources/    — 共享 Database 资源索引
  events/       — 赛事与活动日历
  concepts/     — 可跨辩题迁移的理论概念与方法论
```
